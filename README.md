# 📚 OhneTurn — Nomad Library

> A portable, free, open, public-domain PDF library that travels with you.
> **Read it on any device, anywhere, hands-free.**

This repository is the canonical book source for **OhneTurn** — the hands-free accessibility reader built for Morocco's 1.5 million people living with motor disabilities.

The library is **automatically loaded** when a user opens the OhneTurn app. Every PDF you upload here appears in their sidebar with a 📚 icon. They open it with their gaze, read with text-to-speech, turn pages with a head tilt.

---

## Why "Nomad"?

Books used to live on walls — heavy, fixed, gathering dust. **The Nomad Library has no walls.** It lives on GitHub, mirrors to every OhneTurn user's app, and travels with them across the Maghreb, the Mediterranean, the world. A child in Tetouan, a grandmother in Salé, a student in Casablanca — all see the same shelf, the same books, the same gesture-to-read.

It is a *moving library* — accessible at zero cost, in three languages, with one webcam.

---

## What lives here

- **Public-domain Arabic literature** — Mahmoud Darwish (post-1956 works under fair use), Taha Hussein, Naguib Mahfouz pre-1928, classical poetry
- **Public-domain French literature** — Choukri, Sefrioui (post-50yr works), Camus, Hugo
- **Public-domain English literature** — Project Gutenberg / Internet Archive
- **Moroccan cultural heritage** — folk tales, recipes, history, anything that belongs to *us all*
- **Original team contributions** — student writing, OHNE Official Team essays

## Adding a book

1. Make sure it's **public domain** (or you hold the rights / have explicit permission)
2. Name the file clearly: `Author Name - Book Title.pdf`
*(underscores `_` and dashes `-` both work; OhneTurn formats them automatically)*
3. Organize by language with folders if useful: `/arabic/`, `/french/`, `/english/`
4. Commit & push — the OhneTurn app picks it up on the next launch

The app fetches recursively, so subfolders are automatic.

## Licensing

This repository hosts **only works whose copyright has expired or whose authors have explicitly licensed them for free public distribution.** Each file should ideally include its source / attribution in a sibling `.txt` file or in the PDF's metadata.

If you believe a file infringes copyright, please open an Issue and the maintainers will remove it within 24 hours.

## For libraries, schools, NGOs

If you maintain a collection of Arabic / French / Moroccan public-domain texts and would like them on the Nomad shelf so that disabled readers across Morocco can access them hands-free, **please fork this repo and open a Pull Request**, or contact OHNE Official Team directly.

We are especially looking to partner with:

- **Bibliothèque Nationale du Royaume du Maroc (BNRM)** — institutional partner discussions in progress
- **L'Institut du Monde Arabe (Paris)** — Arabic literary heritage
- **Project Gutenberg** — public-domain English texts
- **Wikisource Arabic** — community-curated Arabic literary corpus
